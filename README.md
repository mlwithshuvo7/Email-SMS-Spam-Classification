## 📌 Project Overview

This project uses **Natural Language Processing (NLP)** and **Machine Learning** to detect spam messages.  
It processes text data, extracts features, and predicts whether a message is spam or not.

---

## ✨ Features

- 📩 Spam vs Ham classification
- 🧠 Machine Learning model (Naive Bayes / others)
- 🔍 NLP preprocessing (tokenization, stemming, stopwords removal)
- 📊 TF-IDF / CountVectorizer feature extraction
- 🌐 Streamlit web interface
- ⚡ Real-time prediction system

---

## 🧠 Machine Learning Workflow
 Data Collection → Text Cleaning → Preprocessing → Vectorization → Model Training → Prediction
